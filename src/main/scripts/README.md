Download JRE and put it into this directory. I recommend OpenJDK.

1. Download OpenJDK for your OS (https://openjdk.java.net/)
2. Extract zip file info subfolder `jre/`
3. It should look like this:

```
europack/
├── jre/
│   └── bin/
│   └── conf/
│   └── include/
│   └── jmods/
│   └── legal/
├── lib/
│   └── *.jar
├──europack.exe
├──europack.jar
└──europack.sh   

```
